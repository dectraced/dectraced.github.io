  Simple _blog + about page_ starter based off of
  [_Ametrine_](https://codeberg.org/daudix/ametrine), a

</span>

> [!CAUTION]
> Not ready for public usage: this theme itself is still in WIP and a tmplate repository can't really help with that.

# Usage and setup
Use this template to generate a new repository with contents identical to this one:  
![Create a repository out of this template](https://github.com/backups1der/assets/blob/main/temp.png?raw=true)

Go to repository settings > Pages tab, then set deployment source to "GitHub Actions"  
![Set deployments to GH Actions](https://github.com/backups1der/assets/blob/main/deploy.png?raw=true)

After waiting ~1 minute, you're all set! Your website now has been built by GitHub Actions
and deployed to GitHub Pages infrastructure. You can access to your website right from the tooltip:  
![Tooltip](https://github.com/backups1der/assets/blob/main/ttip.png?raw=true)

# Extras

In the same page, you can set up custom domains, if you have one and want to use it for this site.
![Set a custom domain](https://github.com/backups1der/assets/blob/main/domain.png?raw=true)

> [!IMPORTANT]
> Do not forget to change/tweak `zola.toml`
> and Markdown files in `/content` as it needs to be filled with your own stuff,
> because content provided here is _just_ enough to get the site built and introduce
> you to some concepts you might not be familiar with.
